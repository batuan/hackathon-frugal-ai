## 1. How to install 
1. Install `uv` from [uv home page](https://docs.astral.sh/uv/getting-started/installation/#installation-methods)
2. Install `python 3.12` or greater with `uv python install 3.12` and then `uv python pin 3.12`
3. `uv sync` to install all packages (include `streamlit + machine learning`)
4. Active python environment with `source .venv/bin/activate`
5. Start `Steamlit` server with `python -m streamlit run main.py`
